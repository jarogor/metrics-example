# metrics-example

## Запуск

```shell
docker-compose up -d
```

## Эндпойнты

- <http://localhost:8000/debug> — для отправки метрик
- <http://localhost:8000/swagger> — автоматически сгенерированная документация по эндпойнтам
- <http://localhost:9090> — Prometheus
- <http://localhost:3000> — Grafana
